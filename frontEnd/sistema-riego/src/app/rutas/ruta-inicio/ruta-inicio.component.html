Sistema de Riego

El Raspberry Pi es un computador del tamaño de una tarjeta de crédito, ejecutado en base a un sistema de código abierto y que posee además de un hardware básico, otro tipo de conexiones como pueden ser las de USB, HDMI, vídeo y sonido, entre otras. Además, se le puede agregar memorias SD, lo que significa que es compatible con un gran número de proyectos, y un punto también importante es que son muy económicos.
